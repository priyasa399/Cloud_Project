Project Name
-------
VNR-GA Elastic Virtual Network Reconfiguration Algorithm Based On Genetic Metaheuristic
Description
------------
This project implements the idea of virtual network reconfiguration  with respect to dynamic growing demands of virtual nodes based genetic algorithm to optimise resource migration,reduce low mapping cost

Requirements:
------------
Language:Python 3.7
Libraries:Network,Graph,numpy
External Tools:Alib
IDE:Google collab

Steps to Run:
------------
1.Keep Alib.zip generate.py graph.py and input.pickle in the same working directory.
2.Install dependencies from Alib.zip that shown in ipynb file
unzip alib-master.zip

3.Install other dependancies gurobipy unidecode networkx numpy
pip install networkx
pip install gurobipy
pip install unidecode
pip install numpy
4.Execute ipynb file
